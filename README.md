# Double linked list example in C++

There is an example of a double linked list that supports multiple types like int, float, double, string, etc.

For this example, it creates three lists: int, float and string.

## Menus

Main menu

    Select a list: 
    [1] int
    [2] float
    [3] string
    [q] quit
    option > 

Action to list

    Using list: int
    Select Action: 
    [1] add
    [2] delete
    [3] clear
    [r] return
    option >
